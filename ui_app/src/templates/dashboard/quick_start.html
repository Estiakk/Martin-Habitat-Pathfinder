Martian Habitat Pathfinder Quick Start Guide
This guide provides a quick overview of how to use the Martian Habitat Pathfinder system. For more detailed information, please refer to the comprehensive user guide.

System Overview

The Martian Habitat Pathfinder is an AI-driven resource management system for Mars habitats. It combines PDF data extraction, NASA data sources, reinforcement learning, and large language models to create a comprehensive simulation and decision-making platform.

Getting Started in 5 Minutes
  1. Upload a PDF: Go to the PDF Management page and upload a scientific paper about Mars habitats.
  2. Process the PDF: The system will automatically extract scientific data and parameters.
  3. Run a Simulation: Go to the Simulation page and start a standard simulation.
  4. View Results: Monitor resource levels and simulation logs to see how the AI manages the habitat.
  5. Try Different AI Modes: Experiment with RL, LLM, and hybrid decision-making approaches.
Key Features
  • PDF Data Extraction: Automatically extract scientific data from research papers.
  • NASA Data Integration: Incorporate real Mars data from NASA sources.
  • Reinforcement Learning: AI agents trained to optimize resource allocation.
  • Large Language Models: Use Ollama to run LLMs for decision-making and explanations.
  • Simulation Environment: Test different scenarios like dust storms and equipment failures.
  • Visualization: Monitor resource levels and trends in real-time.

Troubleshooting

  • Ensure PDFs are not password-protected or encrypted
  • Check that PDFs contain text (not just scanned images)
  • Try processing PDFs individually rather than in batch
  • Check the system logs for specific error messages

  • Ensure Ollama is installed and running on your system
  • Check that the Ollama server is accessible at http://localhost:11434
  • Restart Ollama if it's not responding
  • Make sure you have sufficient disk space for model downloads

  • Reduce the simulation duration for faster results
  • Use RL mode instead of LLM mode for better performance
  • Close other resource-intensive applications
  • Check system resource usage in the dashboard
